Data Visualisation

Three visualisation concepts were developed to working proof of concept stage. Each is delivered as a 'digital first' graphic that is web-native to supports screen-size responsiveness and (better) accessibility. Static versions — that are publication quality vector graphics — can be extracted via the browsers native 'print to PDF' functionality.

A general usage guide has been prepared that details the approach taken to making these graphics more flexible and adaptive to different use cases (e.g. embedding on Vic Gov websites, as exportable static images for presentation, on mobile devices etc.).
