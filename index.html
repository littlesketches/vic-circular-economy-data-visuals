Data Visualisation

Three visualisation concepts were developed to a fully working, proof of concept stage. Each is delivered as a 'digital first' graphic that is web-native to supports screen-size responsiveness and (better) accessibility. Static image versions — that are publication quality vector graphics — can be extracted via the browsers native 'print to PDF' functionality.

A basic user guide has been prepared that details the approach taken to making these graphics more flexible and adaptive to different use cases (e.g. embedding on Vic Gov websites, as exportable static images for presentation, on mobile devices etc.).

1. The System

A representation of the "Shape of the Circular Economy" that focuses on the waste management flows (where data exists).

  • The System: Data graphic summarising the shape of the Victorian Circular Economy as in a system view.

    • Model d: The most dramatic minamalist graphic that focuses on the shape of the waste flows. Available in variants that include an illustration and additional waste recovery breakdown labelling
    • Model T: Version with placeholder space to accommodate the circular economy metrics and explanation (note: design will only be complete if CE Metrics are added). Available in flat variants without the balancing rotation and without the balancing rotation and CE section
    • Model 6:A mirrored version of Model d, with linear flows flowing "down and back".

2. The Journey

A representation of where Victoria has come from and where we are now, in regards to the Circular Economy. This graphic uses data from 2016-2024 (latest) and focuses heavily on trends, events and changes over time.

  • The Journey: In progress data graphic: currently far too crowded and requires curation as to 'what' or 'where' is best ot focus the data story.

3. The Targets

A data graphic that captures Victoria's progress (since 2020) in achieving the four waste targets.

  • The Targets: Proposed data graphic featuring a variant on a sunburst chart to represent progress towards the 2030 targets. In this version, the progress tracks outward from the center.
