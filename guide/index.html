Static export

Some guidance for exporting as a static image

As vector image image (pdf or svg)

Each visualisation is rendered in SVG (the data graphic) and HTML/CSS (for adorning text/headers and components like buttons). These are 'publication quality' vector graphic formats (as opposed to bitmap images) that are: infinitely scalable and do not blur on zoom; much smaller in file size; are fully editable in graphic design software. The most natural export format for web vector graphics is .pdf, which can be achieved by simply using the web browsers Print > to PDF functionality.

By default, each graphic is set to print (in A4) to as either:

  • a landscape page, with adorning text/components located on the left: this can be considered as a 'presentation' layout. Or,
  • a portrait page, with adorning text/components located top and bottom: this can be considered as a 'presentation' layout.

[Technical note]: Static exports in this way use the CSS @print media query to (where possible) format/hide interactive components such as option lists, buttons etc. and replace them with plain text.

The outputted PDF files are designed to be compatible with graphic design software (i.e. editable if being added/tweaked for a publication) and can be dropped into software like MS Word/Powerpoint.

Conversion to bitmap

In cases where a bitmap image (e.g. .png, .jpg etc.) is the only option, the PDF can simply be converted or a screenshot taken.

Website embedding (e.g. on the RV Data Hub)

Two or three visualisation concepts were developed to working proof of concept stage. Each is delivered as a 'digital first' graphic that is web-native to supports screen-size responsiveness and (better) accessibility. Static versions — that are publication quality vector graphics — can be extracted via the browsers native 'print to PDF' functionality.

A general usage guide has been prepared that details the approach taken to making these graphics more flexible and adaptive to different use cases (e.g. embedding on Vic Gov websites, as exportable static images for presentation, on mobile devices etc.).
