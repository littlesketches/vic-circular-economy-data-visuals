# About this repository

This repository comprises of the Data Visualisation Proof of Concept (DataVis PoC) project delivered to Recycling Victoria (RV) when part of the Victorian Department of Energy, Environment and Climate Action (DEECA), in May 2026. 

ALl visualisaitons are delivered in a single web application. A hosted link to the visualisations and documentation contained in this repository can be found online [here](https://vic-circular-economy-data-visuals.vercel.app/).
